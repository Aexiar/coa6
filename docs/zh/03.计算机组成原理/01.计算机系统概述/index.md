# 第一章：课程简介

* 对于下面展示的 `计算机硬件` ，相信很多同学都不会陌生。

![](./assets/image-20251205100607028.png)

* 将 `上述硬件` 安装到 `计算机主板` 的相应 `插座` 或 `插槽` 中，如下所示：

> [!NOTE]
>
> * ① 将 `CPU` 安装到主板上的 `CPU 插座`。
> * ② 将 `内存条` 安装到主板上的 `内存插槽`。
> * ③ 将 `显卡` 安装到主板上的 `PCIE 接口插槽`。
> * ④ 将 `固态硬盘` 安装到主板上的 `M.2 接口插槽`。

![](./assets/image-20251205101105475.png)

* 有过 `装机经历` 的同学一定知道，还需要给 CPU 安装合适的 `风冷散热器` 或 `水冷散热器` ，再给整个 `硬件系统` 选择一个 `功率合适` 的 `电源`，并将 `电源` 配套提供的 `电源线` 连接到 `主板` 上的各种相应的 `供电接口` 以及 `显卡` 上的相应 `供电接口`。

![](./assets/image-20251205101552248.png)

* 之后，将 `上述硬件` 安装到一个 `通风较好` 的` 机箱` ，再配上 `键盘`、`鼠标`、`显示器` 等 `输入输出设备`，就完成了一台计算机硬件的组装。

> [!NOTE]
>
> 地表最强的装机教程：
>
> * ① [硬件茶谈的装机教程](https://www.youtube.com/watch?v=ST65L4pG_6w)。
> * ② [天才赵德柱的装机教程](https://www.youtube.com/watch?v=hX9JmF2bX8w)。

![](./assets/image-20251205101748057.png)

* 至于接下来，就是给 `计算机` 安装 `操作系统` 以及 `各类应用软件` 了，以便我们可以通过计算机解决实际问题。

![](./assets/image-20251205102458947.png)

* 即使很多同学 `没有` 上述 `装机` 经历，很多同学在迈入大学校园之前的那个暑假，也会` 选择` 一台心仪的 `笔记本电脑`，作为自己大学生活的学习或娱乐工具，在选择的过程中，对 `CPU`、`内存`、`显卡`、`硬盘` 等 `计算机硬件`，自然而然的会有一定的了解。

![](./assets/image-20251205102800435.png)

* 即使很多同学，在大一上学期，没有购买自己的台式机或笔记本电脑，也一定会选择一部`智能手机`；因此，对于 `CPU`、`内存`、`显卡`、`硬盘` 等 `计算机硬件`，自然而然的也会有一定的了解。

![](./assets/image-20251205103157466.png)

* 介绍到这里，相信很多同学会产生这样的疑问？

> [!NOTE]
>
> * ① 难道这个课程是用来教如何装机？
> * ② 数码 UP 主的养成？

![](./assets/image-20251205103433329.png)

* 不要误会，之所以进行上述举例，就是为了表达同学们对于 `计算机` 实际上一点都不陌生，因为它是我们生活中不可或缺的工具。但是，对于 `计算机专业的同学` 来说，仅仅知道如何装机、如何安装软件、如何挑选笔记本电脑和智能手机，是远远不够的。我们需要学习的是 `计算机底层的工作原理`，这也就引出了 `《计算机组成原理》` 这门课程会围绕` 哪些问题` 展开。

![](./assets/image-20251205103840498.png)

* 而上述的`这些问题会涉及到哪些计算机硬件`以及`这些硬件又是如何工作的`，我们将在学习 `《计算机组成原理》` 这门课程中，得到解答。

![](./assets/image-20251205104135679.png)

* 对于 `计算机相关专业` 的同学，还需要学习 `《操作系统》`、`《数据结构》`等课程，这样同学们就会对 `整个计算机系统` 有比较深入的理解。

![](./assets/image-20251205104751462.png)

* 在学习了 `《计算机网络》`这门课程后，同学们还会对 `计算机系统之间` 通过 `网络互联` 有一定的理解。

![](./assets/image-20251205105010685.png)

* 那么，上述 `四门课程` 及其 `配套教材`，就是同学们口中的 `四大天书`。

![](./assets/image-20251205105117666.png)

* 然而，这 `四门课程` 也是 `全国硕士研究生入学统一考试` 中 `计算机课程和技术学科` 联考 `指定的课程`，`科目代码` 是 `408` 。

![](./assets/image-20251205105245873.png)
